# 📦 Inventory Management System

Full-stack web application for managing inventory items.

---

## 🚀 Tech Stack

- Frontend: React (Vite)
- Backend: ASP.NET Core Web API
- Database: SQL Server / SQLite

---

## ✨ Features

- Add inventory items (Tablet, Phone, SIM card, Laptop)
- Edit items
- Delete items
- Filter by type, user, and search
- Assign users to items
- Date tracking
- PDF export (templates)

---

## ▶️ How to run

### Backend
- Open project in Visual Studio
- Run API server

API runs on:
http://localhost:5248
